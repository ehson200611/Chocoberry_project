# Загрузка изображений для продуктов

## Настройка завершена ✅

Система загрузки изображений для продуктов полностью настроена.

## Как загрузить изображение через админ-панель:

1. Запустите Django сервер:
```bash
python manage.py runserver
```

2. Откройте админ-панель: http://localhost:8000/admin/

3. Войдите как суперпользователь

4. Перейдите в раздел "Products"

5. Создайте новый продукт или отредактируйте существующий

6. В разделе "Изображение" нажмите "Выберите файл" и загрузите изображение

7. Сохраните продукт

## Как загрузить изображение через API:

### Создание продукта с изображением:

```bash
curl -X POST http://localhost:8000/api/products/ \
  -F "name=Новый продукт" \
  -F "description=Описание" \
  -F "price=99.00" \
  -F "image=@/path/to/image.jpg"
```

### Обновление изображения продукта:

```bash
curl -X PATCH http://localhost:8000/api/products/1/ \
  -F "image=@/path/to/new-image.jpg"
```

## Где хранятся изображения:

- Изображения сохраняются в папке `media/products/`
- URL для доступа: `http://localhost:8000/media/products/filename.jpg`
- API автоматически возвращает полный URL в поле `image_url`

## Форматы изображений:

Поддерживаются все форматы, которые поддерживает Pillow:
- JPEG/JPG
- PNG
- GIF
- WebP
- и другие

## Размер изображений:

Рекомендуется загружать изображения размером не более 2MB для оптимальной производительности.

## Отображение на фронтенде:

- Если у продукта есть загруженное изображение, оно отображается вместо эмодзи
- Если изображения нет, показывается эмодзи по умолчанию
- Изображения автоматически масштабируются и обрезаются для карточек продуктов
